## Polyester
- 에스터 결합을 포함한 고분자
- 에스터기가 연결기가 된 고분자 화합물의 총칭
	- Ester = $RO-C(=O)-R'$ main chain
- 내구성은 강하나 흡습성이 없는 중합체의 일종
- 테레프탈산과 에틸렌글리콜의 축합중합체
- 질기고 변형이 적으며 주름이 잘 안감
- 산에 강하고 열에 약함
## Polyester Fabric
### 특징
- 합성 섬유 중 50%가 넘는 생산량을 보유
- 사계절 모두 사용
- 당겼을 때 나일론 다음으로 강도가 높음
- 물에 젖어도 강도 변화가 없고, 건조 속도가 나일론과 비교했을 때 더 빠름
- 나일론보다 무겁고 신축성이 떨어지나 형태를 유지하는 성질이 있음
### 장점
- 대량생산 가능해 저렴
- 잘 구겨지지 않음
- 염색이 쉽지 않은 대신 이염, 변색, 변형이 잘 되지 않음
- 가공 방법이 많음
