프루닝, lora 파인튜닝을 적용한 모델을 llama.cpp 프레임워크를 이용하여 gguf 포맷으로 변환하는 과정에서 4bit (Q4_K_M) 양자화 기법을 적용하였다. 
Q4_K_M 방식은 가중치(weight)를 4비트 단위로 압축하면서도, 각 블록 단위별로 스케일(scale)과 제로포인트(zero-point)를 유지하여 모델의 연산 정확도를 최대한 보존하는 block-wise quantization 기법이다.
이를 통해 모델의 메모리 사용량과 추론 지연(latency)을 크게 줄이면서도, 정밀도 손실을 최소화할 수 있었다.
결과적으로, 기존 F32 모델 대비 약 4배의 메모리 절감 효과를 얻었으며, 경량 디바이스에서도 안정적인 추론 성능을 확보할 수 있었다.

과정은 다음과 같다. 우선, 앞선 과정에서 저장된 사전 학습된 LoRA 어댑터를 프루닝된 베이스 모델과 병합하여 통합 모델을 생성한다. 이후 llama.cpp를 활용해 통합 모델을 gguf 포맷으로 변환하고, 이후에 4bit 양자화를 적용하였다.
결과적으로, 프루닝, LoRA 기반 파인튜닝, 그리고 양자화가 모두 반영된 최종 경량화 모델이 완성된다.
