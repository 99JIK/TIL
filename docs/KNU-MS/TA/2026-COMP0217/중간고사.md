## 문제 1
연도를 나타내는 양의 정수 n을 입력받아 해당 연도가 윤년인지, 평년인지 출력하시오. (10점)

윤년의 조건은 아래와 같다.
- 4의 배수이면서, 100의 배수가 아닐 때
- 400의 배수일 때
#### 입력 1

```
2000
```
#### 출력 1

```
Leap Year
```
#### 입력 2

```
1900
```
#### 출력 2

```
Common Year
```

```
10 1
```

```
33
```
