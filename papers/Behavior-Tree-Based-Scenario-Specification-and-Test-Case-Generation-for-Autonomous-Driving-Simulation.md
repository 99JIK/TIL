# Behavior-Tree Based Scenario Specification and Test Case Generation for Autonomous Driving Simulation

Behavior Tree(BT)를 활용하여 자율주행 시뮬레이션의 시나리오를 명세하고, 이로부터 테스트 케이스를 자동 생성하는 방법론을 제안한다.

<!-- truncate -->

## 핵심 기여

- **BT 기반 시나리오 명세**: BT의 계층적·모듈적 구조를 이용해 복잡한 교통 시나리오를 재사용 가능한 형태로 기술
- **자동 테스트 케이스 생성**: BT 노드 조합을 통해 다양한 테스트 케이스를 자동으로 생성하는 알고리즘 제안
- **시뮬레이션 연동**: 생성된 테스트 케이스를 자율주행 시뮬레이터에서 실행하여 검증
- **커버리지 향상**: 수동 시나리오 설계 대비 더 광범위한 엣지 케이스 커버리지 달성

## 의의

BT를 단순한 로봇 제어 구조가 아닌 시나리오 명세 언어로 활용한 연구. 자율주행 SIL 테스팅에서 BT의 역할을 확장하는 방향을 제시한다.
