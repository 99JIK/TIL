## 논문 정보
- **제목**: BTGenBot: Behavior Tree Generation for Robotic Tasks with Lightweight LLMs
- **저자**: Riccardo Andrea Izzo, Gianluca Bardaro, Matteo Matteucci (Politecnico di Milano)
- **학회/저널**: arXiv (cs.RO)
- **발행일**: 2025-01-07
- **DOI**: (미제공)
- **주요 연구 내용**: 본 연구는 7B 이하의 매개변수를 가진 경량 거대 언어 모델(LLM)을 로봇 행동 트리(BT) 생성에 활용하는 방안을 탐구한다. GPT-3.5를 사용하여 기존 로봇 BT 데이터에 자연어 설명을 추가한 600개의 학습 데이터셋을 구축하고, Llama2 및 CodeLlama 계열 모델을 파인튜닝하여 성능을 비교 분석한다.
- **주요 결과 및 결론**: 파인튜닝된 경량 모델이 거대한 상용 모델(GPT-4 등)에 근접하는 BT 생성 능력을 갖출 수 있음을 입증한다. 특히 LlamaChat 모델이 구문 및 의미론적 측면에서 가장 우수한 성과를 보였으며, 정적 분석을 통한 후처리가 결과물의 신뢰성을 크게 향상시킨다.
- **기여점**: 로봇 작업을 위한 600개의 비헤이비어 트리-자연어 쌍 데이터셋을 공개하고, 경량 LLM이 로봇 하드웨어에 직접 배포되어 실시간으로 작업을 계획할 수 있는 가능성을 열어준다. 또한 시뮬레이션 및 실제 로봇 환경에서의 다각도 검증 시스템을 제안한다.

## 요약
### 초록
본 논문은 최대 70억 개의 파라미터를 가진 경량 LLM을 사용하여 로봇용 BT를 생성하는 새로운 접근 방식을 제시한다. 특정 데이터셋으로 파인튜닝된 소형 모델이 만족스러운 결과를 낼 수 있음을 증명한다. 연구의 핵심 기여는 GPT-3.5를 활용한 파인튜닝 데이터셋 구축과 Llama2, CodeLlama 등 다양한 모델의 비교 평가이다. 생성된 BT는 정적 분석, 시뮬레이션, 실제 로봇 배포를 통해 철저히 검증한다.

### 서론
현대 로봇공학은 동적이고 비정형화된 환경에서 적응형 작업 계획을 필요로 한다. BT는 복잡한 행동을 구조화하고 확장하는 데 유용한 도구로, ROS2 생태계의 표준으로 자리 잡았다. 기존의 거대 LLM은 강력하지만 막대한 하드웨어 자원을 요구하여 로봇에 직접 배포하기 어렵다는 한계가 있다. 이에 본 연구는 로봇 내부에서 실행 가능한 7B 규모의 경량 모델에 집중한다.

### 배경
거대 언어 모델을 활용한 로봇 계획 연구는 주로 코드를 직접 생성하거나 상용 API를 이용하는 방식에 치우쳐 있다. 하지만 이러한 방식은 데이터 편향이나 하드웨어 제약 문제를 야기한다. 본 연구는 오픈소스 로봇 프로젝트에서 수집된 실제 BT 데이터를 학습에 사용하여 데이터의 신뢰성을 높이고, 로봇 제어에 특화된 XML 형식의 BT 생성을 목표로 한다.

### 모델 아키텍처 / 방법론
- **핵심 구조**: 논문의 **Figure 1**(파인튜닝 과정)에 묘사된 것처럼, 모델 학습은 두 단계로 진행한다. 먼저 Llama-2-7b 기반 모델을 Alpaca 데이터셋으로 명령 이행 학습을 시킨 뒤, 이를 다시 본 연구에서 구축한 BT 전용 데이터셋으로 파인튜닝한다.
- **주요 구성 요소**:
    * **데이터셋**: 600개의 BT와 GPT-3.5로 생성된 자연어 설명을 결합한다. 데이터셋 엔트리는 지시(Instruction), 입력(Input), 출력(Output)의 세 부분으로 구성한다.
    * **검증 시스템**: 생성된 XML의 구문 정확성을 체크하는 정적 분석기와 실제 동작을 확인하는 시뮬레이션/실물 로봇 검증 단계를 포함한다.
- **수식 및 알고리즘**: 효율적인 파인튜닝을 위해 **LoRA**(Low-Rank Adaptation) 기법을 사용한다. 트랜스포머의 셀프 어텐션 모듈에 있는 가중치 행렬 $W_q, W_k, W_v, W_o$ 외에도 일반화 성능 향상을 위해 MLP 레이어의 가중치 행렬들을 추가로 학습시킨다.

### 실험 결과
- **주요 데이터셋 및 설정**: 내비게이션, 조작(Manipulation), 물체 탐색 등 9가지 로봇 작업을 평가 대상으로 설정한다. 하드웨어로는 NVIDIA RTX Quadro 6000 2개와 로봇 배포 테스트를 위한 Jetson AGX Orin을 사용한다.
- **핵심 성능 지표**:
    * **Table I**(성능 기준)에 따르면, GPT-4는 9개 작업 중 7개를 성공한 반면, 기본 Llama 2 13B는 2개 성공에 그쳐 파인튜닝의 필요성을 보여준다.
    * **Table II**와 **Table III**에서 제시된 구문 및 의미론적 평가 결과, 파인튜닝된 모델들이 제로샷(Zero-shot) 환경에서도 기본 모델보다 압도적인 성능 향상을 보인다.
    * **Table V**(2단계 검증)를 통해, 원샷(One-shot) 프롬프트와 정적 분석(SA)을 결합했을 때 LlamaChat 모델이 가장 안정적으로 실행 가능한 BT를 생성함을 확인한다.
    * 논문에 제시된 실제 로봇 플랫폼을 통해 생성된 행동이 물리 환경에서도 유효함을 입증한다.

### 결론
경량 LLM(7B)만으로도 로봇의 복잡한 행동을 계획하고 제어하는 것이 가능하다. 파인튜닝과 적절한 프롬프트 설계는 소형 모델의 한계를 극복하는 핵심 요소이다. 향후 연구는 생성된 행동의 자동 검증 시스템을 강화하고, 사용자의 자연어 입력에 실시간으로 반응하는 인터페이스를 완성하는 데 집중할 계획이다.
